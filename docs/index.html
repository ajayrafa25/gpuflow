GPUFlow v1 — GPU Scheduling
for ML Teams

No Kubernetes. No Slurm. Submit jobs, track experiments, debug in VS Code.

localhost:8000/dashboard
Commands
Submit a job gpuflow run
# image + command, that's it gpuflow run \ --image pytorch/pytorch:2.1.0 \ --gpus 2 --user alice \ torchrun train.py
Run from a path gpuflow run
# code lives on a shared drive gpuflow run \ --image myrepo/env:v3 \ --user bob \ "cd /workspace/bob/proj && torchrun train.py"
Monitor gpuflow status
# see all jobs + who submitted gpuflow status   # stream logs live gpuflow logs <job-id> -f   # cancel gpuflow cancel <job-id>
Experiment tracking MLflow auto
# tracking URI injected automatically # just use mlflow in your script   import mlflow mlflow.log_metric("loss", loss) mlflow.log_param("lr", lr) # → appears in dashboard
